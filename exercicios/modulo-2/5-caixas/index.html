Modelo de caixas

Caixas

O que é uma caixa?

A maioria dos elementos de um HTML são como uma caixa. Containers que armazenam conteúdo ou até mesmo outras caixas.

Anatomia de uma caixa

Os elementos de uma caixa são os listados abaixo:

  • Content(conteúdo): onde está o "bla bla bla"
  • Width e height: Largura e altura, sendo seu conjunto nomeado de "box-size"
  • Border(Borda) fica ao redor do conteúdo
  • padding: espaço entre o conteúdo e a borda
  • Margim(margem): espaço além da borda
  • outline: espaço entre a borda e a margem

Caixas Box-level

O que são?

Caixa do tipo box-level, ao serem adicionadas, pulam uma linha e ocupam uma nova até o fim.

Exemplos

  • <div>
  • <h1>
  • <p>

link

Caixas Inline-level

O que é?

Caixas do tipo inline-level, ao serem adicionadas, não pulam nenhuma linha e só ocupam o espaço necessário para mostrar o conteúdo.

Exemplos

  • <span>
  • <a>
  • <em>
