id e class

Utilizando "id" com CSS

O que é?

O "id" é a identificação de um seletor. Com ele, posso ter duas formatações diferentes em um mesmo tipo de seletor, como dois h1, por exemplo. Recomendado utilizar apenas um por documento.

Como usar?

Para utilizar, basta colocar um "id" após uma tag e dizer qual será seu nome. Após isso, colocar a tag no arquivo CSS com um #(id em CSS) e o nome que você escolheu. Assim, aquela tag se diferenciará das outras O id sempre se sobressairá sobre as classes.

Exemplo em HTML:

<p id="nome"> </p>

Exemplo em CSS:

p#nome{declarações}

Utilizando Class com CSS

O que é?

Basicamente o mesmo que o id, porém, pode-se utilizar vários dentro de um documento e não apenas um.

Como usar?

O mesmo do id, porém, colocando "class" após uma tag e algum nome. Após isso, basta colocar no arquivo CSS um "." (class em CSS) e o nome escolhido (Sem precisar necessariamente colocar a tag antes do ponto, diferente do id). Assim, aquela tag se diferenciará das "padrão".

Exemplo em HTML:

<p class="nome"> </p>

Exemplo em CSS:

.nome{declarações}

Pseudo-classes em CSS

O que é?

É uma palavra-chave adicionada após um sinal de dois pontos em um seletor e especificam o estado de algum elemento (hover, visited, active, empty, etc...).

Como usar?

Para utilizar, basta especificar o elemento e adicionar dois pontos(:). Logo após, adicionar o estado que será alterado.

Exemplo:

link

Pseudo-elementos em CSS

O que é?

Uma palavra-chave adicionada após dois sinais de dois pontos em um seletor e permitem formatar um pedaço específico de um elemento (before, after, first-letter. first-line, etc...)

Como usar?

Para usar, basta especificar o elemento e colocar dois sinais de dois pontos(::). Após isso, adicionar o lugar específico que será alterado.

link

Children(>)

O que é?

Indica que um elemento é "filho" de outro, podendo ser usado para especificar qual elementos gostaríamos de alterar com base em onde ele se encontra

Exemplo


            
 
            <div>
                <p> texto exemplo </p>
            </div>

O "p" está dentro de "div", ou seja, "p" é filho do div, logo div > p
